Web Development Project - Samosa Selector
Submitted by: Duban Correa

This web app: A dynamic "clicker" game built with React where users tap a samosa to earn points and invest in culinary upgrades to increase their clicking power.

Time spent: 5 hours spent in total

Required Features
The following required functionality is completed:

[x] State Management: The app uses React useState to track the total samosa count and the current multiplier.

[x] Interactive Header: Displays the title "Samosa Selector" and the real-time count of samosas earned.

[x] Visual Elements: Features a high-quality samosa image that serves as the primary interaction point.

[x] Upgrade System: Three unique upgrades are available: Double Stuffed (2x), Party Pack (5x), and Full Feast (10x).
d
[x] Responsive Layout: Uses CSS Flexbox to organize the upgrade cards into a clean, horizontal row.

[x] Styled Interface: Implements a dark-themed UI with custom borders, padding, and alignment for a professional look.

Optional (Stretch) Features
The following features are implemented:

[x] Spendable Currency: Purchasing an upgrade reduces the user's total samosa count (Step 5).

[x] Hover Effects: The samosa grows in size when the mouse hovers over it to indicate interactivity (Step 6).

[x] Active Feedback: The samosa "shrinks" slightly when clicked to simulate a physical pulse or button press (Step 6).

[x] Purchase Validation: Logic prevents users from buying upgrades if they do not have enough samosas in their bank.
